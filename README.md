# 🔥 FIRE Planner

A financial independence / retire early calculator. Pure HTML, CSS, and JavaScript — no dependencies, no build step.

## Features

- Multiple savings pots (Pension, ISA, Other) with individual return rates
- Life events modelling (redundancy, career change, etc.)
- UK State Pension toggle
- Retirement drawdown projection with inflation adjustment
- Interactive charts showing wealth growth and drawdown
- FIRE target tracking with milestones

## Deploy

Push to GitHub and enable GitHub Pages (Settings → Pages → Source: main branch, root folder).

Your app will be live at `https://<username>.github.io/<repo-name>/`
